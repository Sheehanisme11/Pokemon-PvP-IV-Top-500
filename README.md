# Great League Bulk IV Finder — Mobile Web App

Static, phone-friendly app generated from your Great League Bulk-First IV database.

- Processed stat entities: 781
- Top 500 rows per entity
- Total ranking rows: 390500
- Level 51 enabled
- Ranking: Defense + HP first, then Attack

Preview locally with: `python -m http.server 8000` then open `http://localhost:8000`.

Upload the folder contents to GitHub Pages, Netlify, or another static host to access it on your phone.
